Chapter3基础语法

3.1类型，变量，与运算符

3.1.1类型

Java可以区分为基本类型和类类型。类类型也叫参考类型。
Java类型主要分为整数，字节，浮点数，字符，布尔。
printf()是JDK5之后才有的API，f的意思是format，也就是格式化。printf()的第一个变量是字符串，其中%d,%n,%d等是格式化的控制符号。

3.1.2 变量

如果在指定变量后，就不想在改变变量值，可以在声明变量时加上final限定。
浮点数除了使用小数方式直接表示外，也可以直接使用科学记号来表示。
JavaSE7之后，撰写整数或者浮点数常量可以使用下划线更清楚的表示么某些数字。
例如：int num=1234_2344;

3.1.3运算符

Java有个条件运算符 条件式？成立返回值：失败返回值

逻辑运算符：&&只要左操作数评估为false，那么会直接返回false，不会再运算右操作符。
对于||运算符，只要左操作符评估为true，那么会直接返回true，不会再运算右操作符。

位运算符：左移运算符会将所有位往左移动指定位数，左移被挤出去的会被丢弃，而右边补上0；右移运算符则相反，会将右移挤出去的位舍弃，至于最左边补上原来的位，如果左边原来是0就补上0，是1就补上1,还有一个>>>运算符，这个运算符在右移时候，做左面一定补上0。

3.1.4类型转换

float PI=3.14f;
float PI=(float)3.14;
如果操作数十不大于int的整数，则会自动提升为Int类型进行计算。

3.2.4 while循环

Math.random()会随机产生0.0到小于1.0的值。
while循环有时被称为前测试循环，因为在循环执行前就进行条件判断。
do…while()又称为后测试循环，先执行动作再判断要不要重复。

3.2.5 break continue

break 和 continue可以配合标签一起使用。
本来break只会离开for循环，但设置标签与区块，就可以离开整个区块。
continue也有类似的用法，不过标签只能设置在for之前。
